📋 Aviso Legal y Política de Privacidad

1. Información General

Nombre de la aplicación: App del III Encuentro Anual de usuarios certificados

Finalidad: Esta aplicación web sirve exclusivamente como herramienta de soporte para el evento presencial, permitiendo a los asistentes acceder a la información del congreso, participar en votaciones anónimas y compartir ideas mediante etiquetas.

Fecha del evento: 5 al 7 de noviembre de 2025

Ubicación: Caixaforum Zaragoza

2. Finalidad de la Aplicación

Esta aplicación tiene las siguientes funcionalidades:

  • Consulta de agenda: Visualización del cronograma de sesiones, ponentes y ubicaciones
  • Votaciones anónimas: Sistema de votación competitivo de ponencias y posters
  • Compartir ideas: Sistema de etiquetas para expresar intereses y necesidades
  • Resultados en tiempo real: Visualización pública de resultados de votaciones
  • Información del evento: Acceso a recursos, materiales y avisos del congreso

🔒 GARANTÍA DE PRIVACIDAD - QUÉ GUARDAMOS Y QUÉ NO:

🟢 EN TU NAVEGADOR (localStorage): Nombre y email (solo local, tú controlas)

🟢 EN NUESTRA BD (Supabase): Solo votos anónimos (vinculados a hash, NO a tu email real) + etiquetas

🔵 EN MICROSOFT FORMS: Datos de registro del evento (gestionado por Microsoft y esPublico, NO por esta app)

❌ NO guardamos en nuestra BD: Nombres, emails, teléfonos ni datos de identificación personal

❌ NO obtenemos: Información de dispositivo, navegador, geolocalización ni seguimiento

❌ NO utilizamos: Cookies de rastreo ni publicidad

❌ NO vinculamos: Tu identidad real con tus votos específicos (anonimato total)

3. Datos Recopilados

La aplicación recopila y almacena los siguientes datos:

A) En tu navegador (localStorage):

  • Nombre completo: Para personalizar tu experiencia (ej: "¡Hola, Ricardo!")
  • Email: Para generar tu código de identificación único
  • Código de identificación: Hash SHA-256 de 16 caracteres derivado de tu email

Estos datos solo existen en tu dispositivo y puedes eliminarlos en cualquier momento cerrando sesión o borrando los datos del navegador.

B) En nuestra base de datos (Supabase):

  • Votos de ponencias: Almacenados vinculados a un hash de tu email (NO al email real)
  • Votos de posters: Almacenados vinculados a un hash de tu email (NO al email real)
  • Etiquetas creadas: Texto libre compartido de forma anónima

Solo se almacenan datos anónimos de interacción. NO guardamos nombres ni emails en nuestra base de datos.

C) Registro de asistencia (Microsoft Forms):

El formulario de registro de asistentes está gestionado por Microsoft Forms, una plataforma externa:

  • Responsable: Microsoft Corporation (servicio Microsoft 365)
  • Datos recopilados: Los que solicite el formulario (nombre, email, organización, etc.)
  • Ubicación: Servidores de Microsoft (cumplimiento RGPD en UE)
  • Política de privacidad: Microsoft Privacy Statement

Importante: Estos datos de registro NO se almacenan en nuestra aplicación ni en nuestra base de datos. Son gestionados exclusivamente por Microsoft Forms y esPublico como organizador del evento.

¿Cómo funciona el sistema de hash para votaciones?

Para garantizar el anonimato en las votaciones, tu email se transforma mediante SHA-256 en un código irreversible de 16 caracteres:

  • ✅ En las tablas de votos NO se guarda tu email real, solo el hash
  • ✅ Nadie puede obtener tu email a partir del hash almacenado
  • ✅ El mismo email siempre genera el mismo hash (para evitar votos duplicados)
  • ✅ Es imposible vincular un voto específico con una persona identificable
  • ✅ Ni siquiera los administradores pueden saber quién votó qué

📊 Resumen de almacenamiento:

  • 🟢 Tu navegador: Nombre + Email + Código hash (solo local, tú controlas)
  • 🟢 Nuestra BD (Supabase): Solo hash anónimo + votos + etiquetas (sin datos personales)
  • 🔵 Microsoft Forms: Datos de registro del evento (gestionado por Microsoft y esPublico)

4. Seguridad de los Datos

La aplicación implementa las siguientes medidas de seguridad:

  • HTTPS: Todas las comunicaciones están cifradas
  • Row Level Security (RLS): La base de datos tiene 46+ políticas de seguridad activas
  • Hashing criptográfico: Los emails se transforman en códigos irreversibles
  • Validaciones automáticas: Triggers que previenen votos duplicados y datos incorrectos
  • Sin almacenamiento de sesión: No hay tokens, cookies de sesión ni información persistente

5. Uso de los Datos

Los datos recopilados se utilizan exclusivamente para:

  • Personalizar tu experiencia en la aplicación
  • Permitir votaciones únicas y anónimas
  • Generar estadísticas agregadas y anónimas del evento
  • Mostrar resultados en tiempo real de votaciones

NO se utilizan para:

  • Marketing, publicidad o prospección comercial
  • Perfilado de usuarios o análisis de comportamiento
  • Venta, cesión o compartición con terceros
  • Comunicaciones posteriores al evento (salvo que lo solicites expresamente)

6. Responsable de los Datos

Datos de votaciones y etiquetas (Supabase):

  • Organizador del evento: esPublico Servicios para la Administración SAU
  • Infraestructura: Supabase (base de datos PostgreSQL en la nube)
  • Ubicación de servidores: Unión Europea (cumplimiento RGPD)
  • Datos almacenados: Solo hash anónimo + votos + etiquetas (sin identificación personal)

Datos de registro de asistentes (Microsoft Forms):

  • Organizador del evento: esPublico Servicios para la Administración SAU
  • Plataforma: Microsoft Forms (Microsoft 365)
  • Responsable del tratamiento: Microsoft Corporation
  • Ubicación de servidores: Microsoft Azure (cumplimiento RGPD en UE)
  • Política de privacidad: https://privacy.microsoft.com/es-es/privacystatement

Importante: Los datos de registro (nombre, email, etc.) son gestionados por Microsoft Forms según sus políticas de privacidad. Nuestra aplicación NO tiene acceso a estos datos de registro.

7. Tus Derechos

Conforme al RGPD (Reglamento General de Protección de Datos), tienes derecho a:

  • Acceso: Consultar qué datos tuyos almacenamos
  • Rectificación: Corregir datos incorrectos
  • Supresión: Solicitar la eliminación de tus datos ("derecho al olvido")
  • Portabilidad: Obtener una copia de tus datos en formato estructurado
  • Oposición: Negarte al procesamiento de tus datos
  • Limitación: Solicitar la restricción del tratamiento

Para ejercer estos derechos, puedes contactar a los organizadores del evento durante el congreso.

8. Retención de Datos

Duración: Los datos se conservarán únicamente durante el evento y hasta 30 días después para generar estadísticas finales.

Eliminación: Pasado este periodo, todos los datos personales serán eliminados de forma irreversible.

Datos agregados: Solo se conservarán estadísticas anónimas (ej: "150 personas votaron", sin identificar a ninguna).

9. Advertencias de Seguridad

⚠️ IMPORTANTE - Recomendaciones de seguridad:

  • No compartas tu pantalla mientras usas la app si no quieres que otros vean tu código de identificación
  • Cierra tu sesión si compartes tu dispositivo con otros asistentes
  • Usa tu email personal, no compartas códigos de acceso con otros
  • Conecta a redes WiFi seguras, evita redes públicas no protegidas si es posible
  • Actualiza tu navegador para tener las últimas medidas de seguridad
  • Los votos son anónimos pero únicos: una vez emitido un voto, no puedes cambiarlo

10. Limitación de Responsabilidad

Esta aplicación se proporciona "tal cual" para uso exclusivo durante el evento. Los organizadores no se responsabilizan de:

  • Pérdida de datos por fallos técnicos o de conexión
  • Uso indebido por parte de terceros
  • Incompatibilidad con navegadores obsoletos
  • Problemas derivados del mal uso de la aplicación

11. Cookies y Tecnologías Similares

Esta aplicación NO utiliza cookies.

Únicamente utiliza localStorage (almacenamiento local del navegador) para mejorar tu experiencia, que no se comparte con ningún servidor externo.
